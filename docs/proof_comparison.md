## 2. 阿贝尔群定理证明

### 数据库中的版本（来自API）

**定理：** 如果对于群 $G$ 任意 $a,b \in G$，都有 $(ab)^{-1} = a^{-1} b^{-1}$，则 $G$ 是阿贝尔群。

**证明：**
要证 $ab=ba$。由 $(ab)^{-1} = a^{-1} b^{-1}$，而一般群 $(ab)^{-1} = b^{-1} a^{-1}$，所以 $a^{-1} b^{-1} = b^{-1} a^{-1}$。对等式两边同时取逆：$(a^{-1} b^{-1})^{-1} = (b^{-1} a^{-1})^{-1}$。根据逆元公式，左边为 $ba$，右边为 $ab$，所以 $ab=ba$。

---

### 原笔记中的版本（TestFile/group.md，第546-578行）

**定理：** Show that a group G is abelian if and only if (ab)^(-1) = a^(-1) b^(-1) for all a,b∈G.

**证明（Step 2部分）：**

If (ab)^(-1) = a^(-1) b^(-1)， then G is abelian

假设条件：
(ab)^(-1)=a^(-1) b^(-1)∀a,b∈G.

(ab)^(-1)=b^(-1) a^(-1)∀a,b∈G.

于是把两个式子放在一起，对任意 a,b∈G有：
b^(-1) a^(-1)=(ab)^(-1)=a^(-1) b^(-1).

所以我们得到：
b^(-1) a^(-1)=a^(-1) b^(-1).

现在对这个等式两边都取逆元（因为在群里，等式两边相等 ⇒ 其逆元也相等）：
(b^(-1) a^(-1) )^(-1)=(a^(-1) b^(-1) )^(-1).

利用你已经证明过的"乘积的逆元"公式：
(xy)^(-1)ⓜ=y^(-1) x^(-1) ║

对左边取 x=b^(-1),y=a^(-1)，对右边取 x=a^(-1),y=b^(-1)，得到：
	左边：(b^(-1) a^(-1) )^(-1)=(a^(-1) )^(-1) (b^(-1) )^(-1)=ab;
	右边：(a^(-1) b^(-1) )^(-1)=(b^(-1) )^(-1) (a^(-1) )^(-1)=ba。

所以
ab=ba∀a,b∈G.
